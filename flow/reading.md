# Reading Flow

After all prompts are complete, produce two artifacts:

1. A material packet.
2. A StoryLens reading.

## Step 1: Material Packet

Use `templates/material-packet.md`.

The packet should include:

- session summary;
- prompt order;
- each prompt title;
- user answer;
- skipped prompts;
- any user context note;
- evidence quality notes.

## Step 2: Evidence Check

Before writing the reading, briefly assess the material:

- Are most prompts answered?
- Are any answers very short?
- Is there enough evidence for repeated patterns?
- Which parts should be treated as low confidence?

Do not shame the user for short answers. Simply name the evidence level.

## Step 3: StoryLens Reading

Use `templates/reading-report.md`.

The reading should include:

- personality signature;
- one-line story signature;
- top signals;
- central tension;
- theme map;
- evidence table;
- relational pattern;
- pressure pattern;
- future orientation;
- type-like language, if useful and tentative;
- counterevidence and limits;
- follow-up questions.

## Step 4: Optional Share Card

If the user asks for something short to save or share, use `templates/share-card.md`.
