# Security Policy

StoryLens Reading Skill is a prompt-and-template package. It does not include a server, user accounts, analytics, payment code, or API keys.

## Reporting Issues

If you find a security or privacy issue in this package, please open a GitHub issue with:

- the affected file;
- what information could be exposed or misused;
- a minimal reproduction if relevant.

Do not include private user story material in public issues.

## User Data Boundary

This package does not collect or store user answers. However, answers entered into an AI assistant may be processed by that assistant's platform. Users should avoid entering highly sensitive details unless they trust the environment they are using.

StoryLens is not a crisis, clinical, hiring, admissions, eligibility, or diagnostic tool.
